创建布局

首先创建一个布局文件：hero.cell.plist：

建议命名规范：[模块].[类型].plist，这里是个UITableViewCell，简写cell

KeyTypeValue
RootDictionary(2 items)
viewsDictionary(2 items)
nameDictionary(1 item)
clazzStringUILabel
descDictionary(1 item)
clazzStringUILabel
constraintsArray(3 items)
Item 0String|-[name]-|
Item 1String|-[desc]-|
Item 2StringV:|-[name]-[desc]-|

其中：

  • views - 待创建的视图列表

    • 各个key将作为视图的别名
    • value作为视图的属性

  • constraints - 约束描述

还有此处尚未配置的 metrics - (边距)尺寸。

这三个配置对应到官方 AutoLayout 的约束API如下：

[NSLayoutConstraint constraintsWithVisualFormat:constrains[i] options:0 metrics:metrics views:views]

有关约束描述语言的细节，可以查看官方文档VFL。

使用布局

在 main.tableview.plist 的 properties 节点下配置行属性 - row：

KeyTypeValue
rowDictionary(2 items)
layoutStringhero.cell
propertiesDictionary(2 items)
@name.textString.$name
@desc.textString.$desc

row 对应 PBRowMapper，将在 PBTableView 的 cellForRowAtIndexPath 代理中将数据映射到指定的cell。

layout 则对应刚才创建的布局文件 hero.cell （不带后缀）。

properties 属性列表，将设置到对应的cell上。

这里，我们注意到：

  • @name 还有 @desc 将对应到 layout 上对应别名的视图。
  • .$name 与 .$desc 则是取到cell对应数据的 name 字段。

我们将在下面用JSON模拟这个数据。

不过现在英雄描述并没有被完整的显示。接下来我们要为表格配置 自动行高。

自动行高

我们在之前配置的约束语句 V:|-[name]-[desc]-| 实际上约定了两个 UILabel 在垂直方向上撑满整个cell。

接下来我们只需要让 @desc 这个 UILabel 自动换行就好，在 hero.cell.plist 的 desc 节点下配置 numberOfLines 为 0:

KeyTypeValue
propertiesDictionary(1 item)
numberOfLinesString0

这里的 properties 属性大家应该不陌生了，所有视图的数据绑定都是通过这个属性。

细节调整

调整单元格分隔线边距

对应系统API：

UITableViewCell.separatorInset = UIEdgeInsets(top, left, bottom, right);

我们让 left 为 0 来让分隔线占满整行，在 main.tableview.plist > row > properties 中进行配置：

KeyTypeValue
separatorInsetString{0,0,0,0}

保存文件：

调整视图间边距

我们来配置一下上文提到的 metrics 来控制各种视图间的边距，在 hero.cell.plist 下增加 metrics 属性：

KeyTypeValue
metricsDictionary(1 item)
leftString48

并在 constraints 中用上 left：

KeyTypeValue
constraintsArray(3 items)
Item 0String|-left-[name]-|
Item 1String|-left-[desc]-|
Item 2StringV:|-[name]-[desc]-|

保存文件，为什么看上去没什么变化？这是因为 Pbind 默认使用 @3x 的基准尺寸，除以3估算为16，刚好是系统默认的左边距。

我们来把 left 改大点看看效果，取两倍 96 吧，保存！

两个 UILabel 间的垂直间距类似可配置，此处就不作演示了。大家可以自己试试看。

基准尺寸通过设定宽度来实现，默认为1080px，你可以自定义为你的项目常用的UI效果图尺寸，如：

[Pbind setSketchWidth:720];
