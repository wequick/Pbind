准备工作: 安装CocoaPods

如果你的电脑还未安装CocoaPods，安装它。

Step 1: 创建与配置工程

在本步骤你将：

  • 创建Application
  • 集成Pbind
  • 集成PBLiveLoader

创建Application

打开Xcode，点选菜单栏 File->New->Project... 创建一个 Single View Application。Product Name 本示例设置为 HeroTour （您可以填写为自己的项目名称）。

集成Pbind

关闭工程，在命令行进入到工程目录，执行命令来初始化Podfile:

pod init

修改生成的Podfile文件，添加依赖pod 'Pbind':

target 'HeroTour' do
  # Uncomment the next line if you're using Swift or would like to use dynamic frameworks
  # use_frameworks!

  # Pods for HeroTour
  pod 'Pbind'

end

执行命令pod install来安装依赖。

集成PBLiveLoader

PBLiveLoader是一个目录监听器，用于即时刷新界面。

首先安装 Pbind 工具集:

gem install pbind

在工程目录下执行命令来集成PBLiveLoader:

pbind watch

成功执行后，工程变动如下：

├── HeroTour
│   └── Info.plist (Mofify)
├── PBLocalhost (Add)
│   └── ignore.h
└── PBLiveLoader (Add)
    ├── PBDirectoryWatcher.h
    ├── PBDirectoryWatcher.m
    ├── PBLiveLoader.h
    └── PBLiveLoader.m

其中：

  • PBLocalhost - 本地服务配置，其下可添加 json 文件来模拟API，ignore.h用以添加例外，配置其中的API将忽略本地，从远程读取。
  • PBLiveLoader - 即时刷新实现，监听 plist, json 文件来响应变更。
  • Info.plist - 监听路径配置，增加了两个字段描述：
    • PBResourcesPath - 当前工程资源路径
    • PBLocalhost - 本地服务配置路径

Step 2: 创建一个基于配置化的界面

重新打开[your-project].xcworkspace。

open HeroTour.xcworkspace

修改 ViewController.h 使其继承至 PBTableViewController

  #import <UIKit/UIKit.h>
  #import <Pbind/Pbind.h>

  @interface ViewController : PBTableViewController


  @end

PBTableViewController持有PBTableView主视图，用于实现TableView的配置化。

修改 ViewController.m 为其指定一个 plist 配置

  - (void)viewDidLoad {
      [super viewDidLoad];
      [self.view setPlist:@"main.tableview"];
  }

新建一个 main.tableview.plist，添加配置如下：

KeyTypeValue
RootDictionary(1 item)
propertiesDictionary(1 item)
headersArray(1 item)
Item 0Dictionary(3 items)
clazzStringUILabel
heightString100
propertiesDictionary(4 items)
textStringHello Pbind!
textColorString#fff
backgroundColorString#333
textAlignmentString:center

如果要复制上述代码，你可以选中上述代码片段的 Root 节点，点击出现的 [COPY] 按钮来复制代码。粘贴时注意先收起 Xcode 的 Root 节点，再选中它，然后按 [⌘+v] 来覆盖。

Step 3: 启动应用

选择一个模拟器，[Command + R] 运行应用。

Step 4: 做一些即时修改

由于没有导航栏，view跟状态栏重叠了，我们通过 margin 来加一点上边距，让界面看起来舒服点：

KeyTypeValue
clazzStringUILabel
heightString100
marginString{100,0,0,0}

[Command + S] 保存文件，界面将立即刷新：

你可以试着改变文字的背景/前景色，来感受下即时刷新带来的畅快开发体验。

更多常量属性的配置可以参考常量表达式与PBRowMapper。
